# Match-Three-Tiles-Game

## Overview

**Match-Three-Tiles-Game** is an engaging number-based tile matching game developed in Java. Players aim to match tiles containing identical hidden numbers within a time limit, with each number uniquely color-coded to enhance visual appeal.

## Gameplay Features

- **Randomised Hidden Numbers**: Each tile conceals a randomly generated number, ensuring a unique experience every game.
- **Color Coding**: Distinct colors represent different numbers, aiding in quick identification and adding vibrancy to the game.
- **Timer Functionality**: Tracks the time taken to form each set, challenging players to improve their speed.
- **Performance Tracking**: Records time elapsed for each set, providing insights into player performance.
- **Endgame Summary**: Displays comprehensive results upon game completion, including total time and sets matched.

## Installation and Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/Jyoti-00/Match-Three-Tiles-Game.git
   cd Match-Three-Tiles-Game
  
2. Open in Eclipse:
Launch Eclipse IDE.
Navigate to File > Open Projects from File System....
Click Directory... and select the cloned repository folder.
Click Finish to load the project.

3. Run the Game:
In the Project Explorer, locate the main class file.
Right-click on the file and select Run As > Java Application.

## How to Play
Start the Game: Launch the application to display the game board with concealed numbers.
Select Tiles: Click on tiles to reveal their hidden numbers.
Match Tiles: Identify and select three tiles with identical numbers to form a set.
Track Time: Aim to match all sets as quickly as possible; the timer records your performance.
Complete the Game: Once all tiles are matched, view your results, including time taken for each set.
